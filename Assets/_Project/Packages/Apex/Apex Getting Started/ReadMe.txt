The demo is divided into three parts.

Part one is available by default and consists of 4 scenes located under the AI Demo folder.
These scenes only rely on Apex Utility AI.

Part two requires Apex Path and consists of 4 scenes located under the AI + Path Demo folder.

Part three requires Apex Steer as well and consists of 3 scenes located under the AI + Path + Steer Demo folder.

All documentation for the various demos/tutorials can be found here: http://apexgametools.com/learn/apex-utility-ai-documentation/apex-utility-ai-getting-started
